# 🏏 IPL Scheduling System

A C-based application that simulates scheduling and management of an IPL-style cricket tournament with real-world constraints.

---

## 🚀 Overview

This project automates IPL match scheduling using a round-robin algorithm while handling constraints such as venue availability, player fatigue, and time-slot optimization.

---

## ✨ Features

- 📅 Automated match scheduling  
- 🏟️ Venue constraint handling  
- ⛔ Restricted dates support  
- 💤 No consecutive matches (player fatigue handling)  
- 📊 Points table with Net Run Rate (NRR)  
- 🌧️ Match cancellation simulation (rain)  
- 🏆 Playoff generation (Qualifier, Eliminator, Final)  

---

## 🛠️ Tech Stack

- Standard Libraries (stdio, time, etc.)
